# FileManagerBundle

Gestionnaire de fichiers pour l’admin AkyosCMS : upload, arborescence, espaces privés, validation MIME et intégration aux formulaires. Dépend de CoreBundle.

## Compatibilité

- PHP >= 8.4
- Symfony ^8.0

## Dépendances Akyos

- `akyos/core-bundle` ^8.0

## Installation

```bash
composer require akyos/file-manager-bundle:^8.0
```
